Основні вимоги до виконання:
Уніфікований контейнер контенту:

Усі сторінки мають містити контент у контейнері з обмеженою шириною та відступами.
Забезпечте наявність відступів від країв контейнера та його центрування на сторінці для кращої візуальної презентації.
Стандартне оформлення спільних елементів:

Загальні блоки (наприклад, меню, футер) повинні мати однаковий дизайн на всіх сторінках.

Структурована форма:

Поля форм повинні бути чітко вирівняні та мати відступи для зручності користувача.
Форми мають займати розумну ширину, враховуючи відступи і не притискатися до країв контейнера.
Логічно організована галерея:

Ваша галерея повинна бути впорядкованою та зручною для перегляду, зображення розташовані по три елементи в один рядок.

Деталі виконання:
Вимоги щодо використання селекторів: Допускається використання тегів, атрибутів та класів в якості селекторів. Прагніть до максимального використання класів для стилізації. Використання ідентифікаторів (ID) для стилізації суворо заборонено.
Застосування навичок стилізації: Використайте вивчені раніше CSS-техніки для створення атрактивного та функціонального дизайну вашого веб-сайту.
Ретельна розробка кожної сторінки: Забезпечте, що кожна сторінка відповідає своїй темі і має відповідний контент та оформлення.
Творчий підхід: Можете експериментувати з дизайном, але не забувайте про основні вимоги задачі.
Відповідність заданому формату: Усі вимоги до структури сайту та його сторінок, які були задані в попередньому домашньому завданні, повинні бути дотримані.
Валідність коду: Переконайтеся, що ваш HTML та CSS код є валідним. Використовуйте відповідні інструменти для перевірки валідності: https://validator.w3.org/#validate_by_upload - HTML валідатор, https://jigsaw.w3.org/css-validator/#validate_by_upload - CSS валідатор.
Форматування коду: Повинно бути виконане форматування коду для його легкого читання та розуміння, бажано щоб було налаштоване автоматичне форматування коду.
Уточнення від ментора:
Ваш ментор може надати додаткові рекомендації чи вимоги до завдання, тому важливо зв'язуватися з ним для уточнень.

Підсумок:
Це завдання дає вам можливість показати ваші навички у створенні та стилізації веб-сторінок. Прагніть до створення якісного та візуально привабливого дизайну, демонструючи розуміння основ веб-розробки та CSS. Успіхів!
